các thành phần web

  1. header - phần đầu
  2. navigation - phần điều hướng (nút bấm ỏ đường link liên kết)
  3. banner - phần ảnh giới thiệu or bìa or trang trí đầu trang
  4. breadcrumb phần đường dẫn kiểu như trong file (ổ G -> HTML )
  5. sidebar là 1 thanh đặt trái ỏ phải màn hình
  6. slider - màn trình diễn các nội dung hình ảnh ,chữ,video
  7. content nội dung trang
  8. footer - chân trang thông thuồng để địa chỉ ỏ logo ỏ thông tin liên hệ
